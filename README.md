# 微信小游戏“跳一跳”autoplay程序

1. 电脑安装adb并加入path
1. Android手机usb连接电脑并允许usb调试、模拟点击
1. 执行`python src/main.py`

## 命令

### 运行

```commandline
python src/main.py
```

### 截图

```commandline
python src/screenshot.py
```

### 检测

```commandline
python src/detect.py
```

### 手动模式

```commandline
python src/hand.py
```
